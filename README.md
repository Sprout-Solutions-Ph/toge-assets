# toge-assets

Public image host for **Toge Design System** email templates (Gmail can only
render images served from public `https://` URLs).

## Usage

Reference images via raw URLs:

```
https://raw.githubusercontent.com/Sprout-Solutions-Ph/toge-assets/main/get-started/card-shapes.png
https://raw.githubusercontent.com/Sprout-Solutions-Ph/toge-assets/main/contributors/jhoko-bernardo.jpeg
```

## Contents

- `get-started/` — logo + the four hero-card images for the onboarding email
- `contributors/` — contributor avatars
